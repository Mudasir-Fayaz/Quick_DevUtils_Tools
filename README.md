# Quick DevUtils

**Quick DevUtils – 150+ Open-Source Tools for Developers**

Quick DevUtils is a powerful web application designed to boost productivity for developers and tech enthusiasts. With over 150 open-source tools, it provides an all-in-one solution for tasks like text manipulation, code formatting, conversions, and more.

Built with a clean and intuitive interface, Quick DevUtils makes it easy to perform complex operations across multiple categories in just a few clicks.

## Tech Stack

Developed using modern technologies for speed and efficiency:

- **Next.js** – Fast and scalable web framework
- **ShadCN & Tailwind CSS** – Elegant and customizable UI components
- **TypeScript** – Type-safe development for reliability
- **Vercel** – Seamless deployment and hosting

Whether you're a developer, designer, or data analyst, Quick DevUtils simplifies your workflow with a comprehensive set of tools—all open source and free to use.🚀

## 🛠️ **Getting Started**

First, clone the repository:

```bash
gh repo clone Mudasir-Fayaz/Quick_DevUtils
# or
git clone https://github.com/Mudasir-Fayaz/Quick_DevUtils_Tools.git
```

Navigate into the project directory and install dependencies:

```bash
cd Quick_DevUtils
pnpm install
```

## Running Locally

Start the development server:

```bash
pnpm dev
# or
npm run dev
# or
yarn dev
# or
bun dev
```

The app will be available at:

👉 **http://localhost:3000**
